# リリースノート (バージョン履歴)

各リリースの変更内容の要約。開発の詳細な経緯は [CHANGELOG.md](CHANGELOG.md) を参照。

## バージョン表記のルール

- `メジャー.マイナー.パッチ` 形式(例: 1.2.3)。
- **機能追加** → マイナーを上げる(1.0 → 1.1 → 1.2 …)。
- **バグ修正・細かい調整** → パッチを上げる(1.0 → 1.0.1 → 1.0.2 …)。
- アプリ内の表示はマーケティング版のみ(例 `v1.0`)。CI実行番号(CFBundleVersion)は表示しない。
- バージョンを上げる操作: `project.yml` の `info.properties.CFBundleShortVersionString` を書き換える(CIがInfo.plistをここから再生成するため。Info.plistファイル単体を直しても効かない)。リリースのたびに本ファイルへ該当バージョンの要約を追記する。

## 1.3.5 (2026-07)

横画面の②③を「画面に表示する行数」で直接制御するように作り替え。

**変更点**

- **②「次の行数」= 画面に入れる行数**を直接指定するように。行の間隔を「画面高さ ÷ 行数」で決めるので、②を少なくすると(文字は横幅上限で頭打ちのぶん)**行間が広くなって行数が減る**。②1で次の行が2行出てしまう問題を解消。
- **③OFF で前の行を確実に隠す**: 現在行を一番上に置く(前の行は画面外)。③ON は現在行を上から2番目にして前の行を1つだけ見せる。②を増やしても③OFFで前行が出てしまう問題を解消。

## 1.3.4 (2026-07)

横画面での②③設定の効きを、実際に体感できるよう調整。

**変更点**

- **③「1行前も表示」を横画面でも有効化**: ONで現在行を画面中央(前の行が上に見える)、OFFで現在行を上寄せ(前の行は画面外、これから来る歌詞が中心)。
- **②「次の行数」でサイズが必ず変わるように**: 以前は②が小さいと文字サイズが上限に張り付いて変化が見えなかったのを、②=1〜5で段階的にサイズ+行間が変わるよう修正(②=1は大きく行数少なめ、②=5は小さく行数多め)。

## 1.3.3 (2026-07)

横画面で、1行の歌詞が2行に折り返してしまう問題を調整。

**変更点**

- 横画面の歌詞は**各行を1行に固定**し、幅からはみ出す長い行は折り返さずに縮小して収めるようにした(特に「次の行数=1・1行前OFF」で文字が大きくなりすぎて折り返していた状態を解消)。
- 文字サイズの上限を下げ、横幅に対して大きすぎない値に調整。

## 1.3.2 (2026-07)

横画面の全画面歌詞を、全行スクロール表示に変更した。

**変更点**

- 横画面の歌詞を、現在行を中央に保ちながら曲の進行に合わせて上へ流れる**全行スクロール**に(縦画面や一般的な歌詞アプリと同じ、読みやすい挙動)。
- 表示設定の「次の行数(②)」「1行前も表示(③)」は、横画面では**文字サイズに読み替えて適用**。行数を少なくすると文字が大きく(没入)、多くすると小さく(文脈重視)なる。
- PiPは従来どおり②③がそのまま(小窓に何行)適用される(変更なし)。

## 1.2 (2026-07)

横画面での全画面歌詞表示を追加。

**変更点**

- アプリを開いたまま**横向きにすると、全画面で歌詞を表示**するようになった。表示内容・設定(行数・1行前表示)はPiPと同じものを反映。
- PiPと違い**アプリ自身の画面**なので、iOSのサイズ・位置の制約を受けず、大きな文字で表示できる。縦に戻すと通常画面に戻る。
- バックグラウンド時のPiP表示は従来どおり(この機能はPiPの置き換えではなく追加)。
- 補足: 横画面での**PiP**を画面いっぱいに広げる案は、PiPウィンドウの大きさ・位置をアプリから指定できないiOSの制約により断念。代わりに本機能(アプリ内の横画面全画面表示)で対応した。

## 1.1 (2026-07)

PiPに表示する歌詞の行数を設定で変えられるようにした。

**変更点**

## 1.0 (2026-07)

初回リリース。Spotifyで再生中の曲に同期して歌詞をPiP(Picture in Picture)でフロート表示するiPhoneアプリ。

**主な機能**

- Spotifyで再生中の曲を**自動検知**(Spotify内部のリアルタイム経路を使用。公式Web APIは使わない)。曲の切り替え・一時停止にも追従。
- **同期歌詞**を表示。Spotify自身の同期歌詞を優先し、無ければlrclibから取得。
- **PiPで歌詞をフロート表示**。完全自動で、アプリをバックグラウンドに回すと表示・フォアグラウンド復帰で終了(手動ボタンなし)。
- 横長のPiPウィンドウに2行表示(現在行を強調 + 次の行)。長い行はフォントを自動縮小して収める。
- プレイリスト再生・広告の挟み込みにも対応して動作を継続。

**前提・制約**

- Spotify Premium不要・追加コスト不要・マイク不使用。
- 個人利用向け(App Store配布なし、SideStoreでサイドロード)。
- lrclib/Spotifyのどちらにも同期歌詞が無い曲は「歌詞なし」表示になる。
