# リリースノート (バージョン履歴)

各リリースの変更内容の要約。開発の詳細な経緯は [CHANGELOG.md](CHANGELOG.md) を参照。

## バージョン表記のルール

- `メジャー.マイナー.パッチ` 形式(例: 1.2.3)。
- **機能追加** → マイナーを上げる(1.0 → 1.1 → 1.2 …)。
- **バグ修正・細かい調整** → パッチを上げる(1.0 → 1.0.1 → 1.0.2 …)。
- アプリ内の表示は `v{バージョン} (build {CI番号})`。build番号はGitHub Actionsの実行番号で、どのCIビルドが入っているかの識別用。バージョン(マーケティング番号)とは別物。
- バージョンを上げる操作: `LyricsPiP/App/Info.plist` の `CFBundleShortVersionString` を書き換える。リリースのたびに本ファイルへ該当バージョンの要約を追記する。

## 1.0 (2026-07)

初回リリース。Spotifyで再生中の曲に同期して歌詞をPiP(Picture in Picture)でフロート表示するiPhoneアプリ。

**主な機能**

- Spotifyで再生中の曲を**自動検知**(Spotify内部のリアルタイム経路を使用。公式Web APIは使わない)。曲の切り替え・一時停止にも追従。
- **同期歌詞**を表示。Spotify自身の同期歌詞を優先し、無ければlrclibから取得。
- **PiPで歌詞をフロート表示**。完全自動で、アプリをバックグラウンドに回すと表示・フォアグラウンド復帰で終了(手動ボタンなし)。
- 横長のPiPウィンドウに2行表示(現在行を強調 + 次の行)。長い行はフォントを自動縮小して収める。
- プレイリスト再生・広告の挟み込みにも対応して動作を継続。

**前提・制約**

- Spotify Premium不要・追加コスト不要・マイク不使用。
- 個人利用向け(App Store配布なし、SideStoreでサイドロード)。
- lrclib/Spotifyのどちらにも同期歌詞が無い曲は「歌詞なし」表示になる。
